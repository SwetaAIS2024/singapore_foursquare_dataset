THE FOURSQUARE PLACES (SINGAPORE) DATASET

This dataset has data on 340097 points of interests (POI) in Singapore belonging to 953 unique categories. It has been extracted from the Foursquare Open Source Places Dataset [1][2], a worldwide dataset with over 100 million POIs.
